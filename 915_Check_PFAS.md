## 3. Percentage over LOQ, all stations    

### Table    

```r
df1 <- data_all %>% 
  filter(substr(PARAM, 1, 2) == "PF") %>%
  group_by(PARAM, MYEAR) %>%
  summarise(
    Over_LOQ = round(100*mean(is.na(FLAG1)),0)
    )# regrouping output by 'PARAM'
```


```r
# For correct oredring of PARAM
```
